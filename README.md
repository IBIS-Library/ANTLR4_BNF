# ANTLR4_BNF
ANTLR4 development files

These files are provided to permit experimentation with ANTLR 4 (antlr.org) and the original BNF for IBIS 2.0 defined by Bob Ross et al in 1994.

Note that only the IBIS header section is working reliably.  Editing and updating are strongly encouraged.

More information is available through the IBIS Summit presentation under https://ibis.org/summits/feb26/mirmak.pdf
